# 周六模板详细编辑功能使用指南

## 🎯 功能概述

新增的周六模板详细编辑功能允许您精细化控制每个服事角色的到岗时间和具体说明，实现更灵活的通知模板管理。

## 🚀 如何使用

### 1. 进入编辑界面

1. 启动应用：`streamlit run app_unified.py`
2. 在侧边栏选择 **"🛠️ 模板编辑器"**
3. 在 **"📝 编辑模板"** 标签页中选择 **"🔔 周六提醒通知"**

### 2. 三个编辑标签页

#### 📝 主模板
- **主模板内容**：编辑整体通知结构
- **Detail格式模板**：控制每个角色detail的显示格式
  - 可用变量：`{person}`（人员名字）、`{time}`（到岗时间）、`{instruction}`（具体说明）
- **无人员时的格式**：当某个角色没有安排人员时的显示格式

#### ⏰ 时间配置
- 为每个服事角色设置具体的到岗时间
- 提供三种预设模式：
  - **🎵 敬拜排练模式**：音控和PPT播放9:00，导播9:30，PPT更新提前
  - **🕐 统一时间**：所有角色统一设为9:00
  - **🔄 重置默认**：恢复系统默认时间设置

#### 📋 说明配置
- 为每个服事角色设置具体的工作说明
- 提供预设模式：
  - **🎵 敬拜配合模式**：设置与敬拜团配合的标准说明
  - **🔄 重置默认**：恢复系统默认说明

### 3. 预览效果

在 **"👁️ 预览效果"** 标签页中可以：
- 查看有人员安排和无人员安排时的不同效果
- 查看当前配置摘要
- 使用实时预览工具测试不同的参数组合

## 📋 配置示例

### 当前默认配置

**到岗时间：**
- 音控：9:00
- 导播/摄影：9:30
- ProPresenter播放：9:00
- ProPresenter更新：提前

**具体说明：**
- 音控：随敬拜团排练
- 导播/摄影：检查预设机位
- ProPresenter播放：随敬拜团排练
- ProPresenter更新：提前准备内容

### 生成效果示例

```
【主日服事提醒】✨
明天 8:30布置/ 9:00彩排 / 10:00 正式敬拜
请各位同工提前到场：

- 音控：Jimmy 9:00到，随敬拜团排练
- 导播/摄影：靖铮 9:30到，检查预设机位
- ProPresenter播放：张宇 9:00到，随敬拜团排练

愿主同在，出入平安。若临时不适请第一时间私信我。🙌
```

## 🔧 高级自定义

### Detail格式模板语法

默认格式：`{person} {time}到，{instruction}`

您可以自定义为：
- `{person} 请于{time}到场，{instruction}`
- `{person}同工：{time} - {instruction}`
- `【{person}】{time}到岗，注意事项：{instruction}`

### 变量说明

- `{person}`：会被实际的人员名字替换（如"Jimmy"、"靖铮"）
- `{time}`：会被设置的到岗时间替换（如"9:00"、"9:30"）
- `{instruction}`：会被具体说明替换（如"随敬拜团排练"）

## 💾 保存和同步

编辑完成后：
1. 点击对应的保存按钮（主模板、时间配置、说明配置可以分别保存）
2. 在 **"💾 保存管理"** 标签页中选择保存到本地或云端
3. 配置会自动应用到后续的模板生成中

## 🎯 使用场景

- **特殊主日**：某些主日需要提前更多时间到场
- **设备调试**：新设备需要额外的调试时间
- **人员培训**：新同工需要更详细的说明
- **节期安排**：圣诞节、复活节等特殊安排
