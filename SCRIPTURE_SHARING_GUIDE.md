# 📖 经文分享功能使用指南

## 功能概述

经文分享功能已成功集成到Grace Irvine Ministry Scheduler系统中，在每次生成周三确认通知时会自动包含一段经文分享。

## ✨ 主要特性

- 📚 **经文库管理**: 支持添加、编辑、删除经文内容
- 🔄 **自动轮换**: 每次生成通知时按顺序选择下一段经文
- 📝 **智能格式化**: 自动格式化经文显示，包含出处、内容和主题
- ☁️ **云端同步**: 支持本地和云端存储，配置自动同步
- 🎯 **主题分类**: 每段经文可设置主题和备注说明

## 📋 使用方法

### 1. 访问经文管理界面

1. 启动应用: `python3 -m streamlit run app_unified.py`
2. 在侧边栏选择 "📖 经文管理"
3. 进入经文管理页面

### 2. 管理经文内容

#### 📝 查看经文列表
- **经文列表**标签页显示所有经文
- 当前要使用的经文会标记为"下一个使用"
- 显示完整的经文内容（包含经文和出处）

#### ➕ 添加新经文
1. 切换到**添加经文**标签页
2. 按照指定格式粘贴经文内容：
   ```
   看哪，弟兄和睦同居
   是何等地善，何等地美！
   (诗篇 133:1 和合本)
   ```
3. 点击"📝 添加经文"

#### 🔧 编辑现有经文
1. 切换到**编辑经文**标签页
2. 从下拉菜单选择要编辑的经文
3. 按照格式要求修改经文内容
4. 点击"💾 保存修改"

#### 🗑️ 删除经文
- 在经文列表中，点击对应经文的"🗑️ 删除"按钮

### 3. 预览和测试

#### 📊 使用预览标签页
- **当前经文预览**: 显示即将使用的经文格式
- **下一经文预览**: 显示下次将使用的经文
- **模板集成预览**: 测试完整的周三通知模板

#### 🧪 测试生成
点击"🧪 测试周三通知模板"查看包含经文的完整通知内容

### 4. 通知模板生成

#### 📝 在模板生成器中使用
1. 选择"📝 模板生成器"页面
2. 选择"周三确认通知"
3. 生成的通知会自动包含当前经文分享

#### 示例输出格式
```
【本周9月17日主日事工安排提醒】🕊️

• 音控：Jimmy
• 导播/摄影：靖铮
• ProPresenter播放：张宇
• ProPresenter更新：Daniel
• 视频剪辑：靖铮

📖 看哪，弟兄和睦同居
是何等地善，何等地美！
(诗篇 133:1 和合本)

请大家确认时间，若有冲突请尽快私信我，感谢摆上 🙏
```

## 🔧 管理操作

### 索引管理
- **🔄 重置索引**: 将经文选择位置重置到第一段
- **⏭️ 跳过当前**: 跳过当前经文，移动到下一段
- **💾 保存配置**: 手动保存经文配置

### 轮换机制
- 经文按照添加顺序循环使用
- 每次生成通知后自动移动到下一段
- 到达最后一段后自动回到第一段

## 📁 文件结构

- **经文配置**: `templates/scripture_sharing.json`
- **模板配置**: `templates/dynamic_templates.json`
- **经文管理器**: `src/scripture_manager.py`
- **模板管理器**: `src/dynamic_template_manager.py`

## 🔍 故障排除

### 常见问题

1. **经文不显示**
   - 检查 `templates/scripture_sharing.json` 文件是否存在
   - 确认经文库中有内容

2. **经文不更新**
   - 点击"💾 保存配置"确保更改已保存
   - 重启应用重新加载配置

3. **模板错误**
   - 检查模板中是否包含 `{scripture_sharing}` 变量
   - 在模板编辑器中验证模板语法

### 重新初始化

如果遇到问题，可以删除 `templates/scripture_sharing.json` 文件，系统会自动创建默认配置。

## 🛠️ 技术细节

### 数据结构
```json
{
  "metadata": {
    "version": "1.0",
    "current_index": 0,
    "total_count": 10
  },
  "scriptures": [
    {
      "id": 1,
      "content": "看哪，弟兄和睦同居\n是何等地善，何等地美！\n(诗篇 133:1 和合本)"
    }
  ]
}
```

### API接口
- `get_next_scripture()`: 获取并更新到下一段经文
- `get_current_scripture()`: 获取当前经文（不更新索引）
- `add_scripture(content)`: 添加新经文（只需经文内容）
- `update_scripture(id, content)`: 更新现有经文（只需经文内容）
- `delete_scripture(id)`: 删除经文
