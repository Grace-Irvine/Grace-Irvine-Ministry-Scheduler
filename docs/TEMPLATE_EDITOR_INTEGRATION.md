## 🚀 使用方法

### 1. 启动应用
```bash
streamlit run streamlit_app.py
```

### 2. 访问模板编辑器
- **方法1**：在侧边栏选择"🛠️ 模板编辑器"页面
- **方法2**：点击侧边栏的"🛠️ 快速编辑模板"按钮

### 3. 编辑模板
1. 在"编辑模板"选项卡中选择模板类型
2. 修改模板内容和默认值
3. 点击"应用更改"保存到内存
4. 点击"保存到文件"永久保存

### 4. 预览效果
1. 切换到"预览效果"选项卡
2. 查看有安排和无安排时的渲染效果
3. 参考变量说明进行调整

### 5. 真实测试
1. 切换到"真实测试"选项卡
2. 查看使用实际Google Sheets数据的效果
3. 对比三种通知类型的生成结果
