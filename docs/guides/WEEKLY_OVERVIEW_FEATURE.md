# Grace Irvine Ministry Scheduler - 周程概览功能

## 🎯 功能概述

新增的**周程概览功能**为数据概览页面添加了强大的时间维度分析，让您能够清晰地看到过去、现在和未来的事工安排情况。

## ✨ 主要特性

### 📅 时间范围覆盖
- **前两周**: 查看已完成的事工安排
- **当前周**: 重点关注本周主日安排
- **未来7周**: 提前规划未来的事工需求

### 🎯 智能时间标签
- **前两周** / **上周** - 历史安排回顾
- **本周** - 当前重点关注
- **下周** / **下下周** - 近期安排
- **未来第X周** - 长期规划

### 📊 可视化展示
- **表格形式**: 清晰的行列布局
- **状态标识**: 已完成/本周/未来状态
- **角色分列**: 音控、导播/摄影、ProPresenter播放/更新
- **统计指标**: 已安排周数、完成情况、待进行数量

## 🔔 智能提醒系统

### 🚨 紧急提醒
- **本周主日无安排**: 红色警告提示
- **关键角色缺失**: 音控、导播、ProPresenter 未安排

### ⏰ 预警提醒
- **下周安排不完整**: 黄色警告，建议提前准备
- **未来周次空白**: 提前规划提醒

### 👥 冲突检测
- **重复安排**: 同一人在同一主日担任多个角色
- **人员分布**: 志愿者工作负荷分析

## 🚀 技术实现

### 核心算法
```python
def get_sunday_of_week(target_date):
    """获取指定日期所在周的周日日期"""
    days_since_sunday = target_date.weekday() + 1
    if days_since_sunday == 7:  # 如果是周日
        days_since_sunday = 0
    return target_date - timedelta(days=days_since_sunday)

def get_week_label(sunday, today):
    """智能生成周次标签"""
    current_sunday = get_sunday_of_week(today)
    week_diff = (sunday - current_sunday).days // 7
    
    if week_diff == -2: return "前两周"
    elif week_diff == -1: return "上周"
    elif week_diff == 0: return "本周"
    elif week_diff == 1: return "下周"
    # ... 更多逻辑
```

### 数据处理流程
1. **时间计算**: 基于当前日期计算目标周日期
2. **数据分组**: 按周日对排程数据进行分组
3. **状态判断**: 根据日期关系确定状态
4. **智能提醒**: 分析数据生成相关提醒
