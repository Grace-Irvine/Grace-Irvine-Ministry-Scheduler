# 新 ICS 前端查看器使用指南

## 📋 功能概述

前端已添加专门的多类型 ICS 日历查看器，方便查看和调试云端生成的 ICS 文件。

## 🎯 使用方法

### 1. 访问前端界面

启动 Streamlit 应用：
```bash
streamlit run app_unified.py
```

或使用：
```bash
python start.py
```

### 2. 进入 ICS 日历管理页面

在侧边栏选择 **"📅 ICS日历管理"** 页面

### 3. 使用多类型 ICS 日历查看器

在页面底部，你会看到 **"🆕 多类型 ICS 日历查看器"** 部分

#### 功能特性：

1. **数据源选择**
   - 🌐 智能读取（云端优先）- 自动选择最佳数据源
   - 💻 本地文件 - 仅从本地读取
   - ☁️ 仅云端 - 仅从 GCS bucket 读取

2. **ICS 类型选择**
   - 🎤 媒体部服事日历 - `media-team.ics`
   - 👶 儿童部服事日历 - `children-team.ics`
   - 📋 每周全部事工概览 - `weekly-overview.ics`

3. **查看功能**
   - 📅 事件列表 - 显示所有事件，支持搜索和筛选
   - 📊 统计分析 - 事件统计、时间分布等
   - 🔧 原始数据 - 查看原始 ICS 文件内容
   - 📖 事件详情 - 按日期分组显示事件

## 🔍 调试功能

### 查看事件描述（包含经文）

在 **"📅 事件列表"** 标签页中：
- 每个事件都可以展开查看详细描述
- 描述中包含：
  - 服事安排信息
  - **经文分享内容**（原来功能附加的经文）
  - 通知模板格式

### 搜索和筛选

- 使用搜索框可以按关键词筛选事件
- 支持搜索标题和描述内容

### 查看原始 ICS 内容

在 **"🔧 原始数据"** 标签页中：
- 显示完整的 ICS 文件内容
- 可以下载 ICS 文件
- 验证 ICS 格式

### 按日期分组查看

在 **"📖 事件详情"** 标签页中：
- 所有事件按日期分组
- 方便查看同一天的所有通知事件

## 🔄 生成 ICS 文件

如果文件不存在，可以直接在前端生成：

1. 选择要查看的 ICS 类型
2. 如果文件不存在，会显示生成按钮
3. 点击 **"🔄 生成 [类型]"** 按钮
4. 系统会自动生成并保存文件

## 💡 调试技巧

1. **查看经文内容是否正确**
   - 在事件列表中展开第一个事件
   - 检查描述中是否包含 `📖` 开头的经文内容

2. **验证通知时间**
   - 在事件详情标签页中按日期分组查看
   - 确认周三、周六、周一的通知时间是否正确

3. **检查数据源**
   - 查看页面顶部显示的数据源信息
   - 确认是从云端还是本地读取

4. **对比不同类型**
   - 可以同时在不同浏览器标签页打开
   - 对比媒体部、儿童部、每周概览的差异

## 🐛 常见问题

### 问题 1: 文件不存在
**解决方案**: 点击生成按钮，系统会自动生成文件

### 问题 2: 无法读取云端文件
**解决方案**: 
- 检查 GCS 连接状态
- 尝试选择 "💻 本地文件" 模式
- 检查服务账号权限

### 问题 3: 事件数量为 0
**解决方案**:
- 检查数据源中是否有未来的日期
- 检查配置中的日期范围设置
- 查看日志了解详细错误
