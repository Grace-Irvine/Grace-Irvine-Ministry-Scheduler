# 📧 模板邮件发送功能

## 🎯 功能简介

在UI模板生成器中新增了邮件发送功能，让您可以一键将生成的微信群通知模板发送到指定邮箱，方便事工负责人查看和转发。

## ✨ 新增功能

### 1. 模板生成器页面
在"📝 模板生成器"页面中，生成模板后可以：
- **💾 保存模板到文件**：保存到本地文件
- **📧 发送到邮箱**：发送精美的HTML邮件
- **📋 复制提示**：提醒手动复制内容

### 2. 模板编辑器页面
在"🛠️ 模板编辑器"的"🧪 真实测试"选项卡中，每种通知都有：
- **📋 复制提示**：提醒手动复制
- **📧 发送邮件**：直接发送到邮箱

## 📧 邮件特色

### 1. 精美的HTML设计
- **响应式布局**：支持桌面和移动端查看
- **教会品牌色彩**：使用Grace Irvine教会的视觉风格
- **清晰的结构**：头部、内容、使用说明、页脚

### 2. 完整的使用指导
- **步骤说明**：详细的复制粘贴指导
- **发送建议**：最佳发送时间提醒
- **格式保持**：保持微信群通知的原始格式

### 3. 智能邮件主题
根据模板类型和服事日期自动生成邮件主题：
- `【微信群通知模板】周三确认通知 - 8月31日`
- `【微信群通知模板】周六提醒通知 - 9月1日`
- `【微信群通知模板】月度总览通知`

## 🚀 使用方法

### 方法1：通过模板生成器
1. 进入"📝 模板生成器"页面
2. 选择模板类型（周三确认/周六提醒/月度总览）
3. 查看生成的模板内容
4. 点击"📧 发送到邮箱"按钮
5. 等待发送完成，查看状态提示

### 方法2：通过模板编辑器
1. 进入"🛠️ 模板编辑器"页面
2. 切换到"🧪 真实测试"选项卡
3. 查看使用真实数据生成的三种通知
4. 在任意通知下方点击"📧 发送邮件"按钮
5. 等待发送完成

## ⚙️ 配置要求

### 1. 环境变量配置
确保 `.env` 文件中配置了以下变量：
```bash
SENDER_EMAIL=jonathanjing@graceirvine.org
EMAIL_PASSWORD=your_app_password
GOOGLE_SPREADSHEET_ID=your_sheet_id
```

### 2. 收件人设置
默认发送给 `SENDER_EMAIL` 配置的邮箱，如需修改：
- 在 `send_template_email` 函数中调整收件人设置
- 可以添加多个收件人或从配置文件读取
