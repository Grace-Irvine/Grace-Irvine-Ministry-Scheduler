# 个人ICS系统更新说明

## 📅 更新日期：2025-10-03

## 🔄 修改内容

### 1. 视频剪辑截止时间调整
**修改前**：主日后的周一晚上8点  
**修改后**：主日后的周二晚上8点

**原因**：给予更多时间完成剪辑工作

**示例**：
- 10/05 (周日) 主日 → 10/07 (周二) 20:00 截止
- 10/12 (周日) 主日 → 10/14 (周二) 20:00 截止

### 2. 取消彩排事件
**修改前**：为每个服事角色创建两个事件
- 彩排事件（到场时间 → 10:00）
- 正式服事事件（10:00 → 12:00）

**修改后**：只创建一个服事事件
- 服事事件（到场时间 → 12:00）

**原因**：简化日历，减少重复事件

### 3. 服事时间调整

#### 音控
- **到场时间**：09:00
- **结束时间**：12:00
- **时长**：3小时

#### ProPresenter播放
- **到场时间**：08:30
- **结束时间**：12:00
- **时长**：3.5小时

#### 摄像导播
- **到场时间**：09:30
- **结束时间**：12:00
- **时长**：2.5小时

#### ProPresenter更新
- **到场时间**：08:30
- **结束时间**：12:00
- **时长**：3.5小时

#### 视频剪辑
- **无到场要求**（远程工作）
- **截止时间**：主日后周二晚8点

### 4. 提醒设置调整

#### 服事事件（所有现场角色）
**修改前**：
- 彩排提醒：提前60分钟
- 服事提醒：提前30分钟

**修改后**：
- 提醒1：提前30分钟
- 提醒2：开始时（0分钟）

#### 视频剪辑
**保持不变**：
- 提醒：提前1天（1440分钟）

### 5. 地点更新
**修改前**：Grace Irvine 教会  
**修改后**：200 Cultivate, Irvine, CA 92618

## 🔧 技术细节

### 视频剪辑截止时间计算
```python
# 计算主日后的周二
days_until_tuesday = (2 - service_date.weekday()) % 7
if days_until_tuesday == 0:
    days_until_tuesday = 2
deadline_date = service_date + timedelta(days=days_until_tuesday)
```

### 双重提醒实现
```python
# 提醒1：提前30分钟
alarm1 = Alarm()
alarm1.add('trigger', timedelta(minutes=-30))

# 提醒2：开始时
alarm2 = Alarm()
alarm2.add('trigger', timedelta(minutes=0))
```
