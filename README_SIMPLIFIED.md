# Grace Irvine Ministry Scheduler - 简化版

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red.svg)](https://streamlit.io/)

**恩典尔湾长老教会事工排程管理系统 - 简化版**

## 🎯 功能概述

这是一个简化的事工排程管理系统，专注于核心功能：

### ✨ 核心功能
- **📊 数据查看**: 从 Google Sheets 自动提取和清洗数据
- **📝 模板生成**: 自动生成微信群通知模板
- **🔍 数据分析**: 实时数据质量监控和统计
- **📥 数据导出**: 支持 Excel 和 CSV 格式导出

### 🚀 简化架构
```
简化版架构:
├── streamlit_app.py          # Web 界面主应用
├── simple_data_cleaner.py    # 数据清洗核心模块
├── run_streamlit.py          # 启动脚本
├── requirements.txt          # 依赖包列表
└── data/                     # 数据存储目录
```

## 📋 快速开始

### 1. 环境准备

```bash
# 安装依赖包
pip3 install -r requirements.txt
```

### 2. 配置 Google Sheets

1. 确保您的 Google Sheets 设置为"知道链接的任何人都可以查看"
2. 从 URL 中提取 Spreadsheet ID
3. （可选）创建 `.env` 文件设置默认 ID：
   ```
   GOOGLE_SPREADSHEET_ID=1wescUQe9rIVLNcKdqmSLpzlAw9BGXMZmkFvjEF296nM
   ```

### 3. 启动 Web 应用

```bash
# 方式1: 使用启动脚本（推荐）
python3 run_streamlit.py

# 方式2: 直接启动 Streamlit
streamlit run streamlit_app.py
```

应用将在浏览器中自动打开：http://localhost:8501

## 🛠️ 故障排除

### 常见问题

1. **无法连接 Google Sheets**
   - 确保表格设置为公开访问
   - 检查 Spreadsheet ID 是否正确
   - 验证网络连接

2. **数据质量低**
   - 检查原始表格数据格式
   - 确认日期列格式正确
   - 验证人名数据完整性

3. **模板生成失败**
   - 确认表格中有对应日期的数据
   - 检查列名是否包含关键词（音控、导播等）
   - 验证数据清洗结果

### 解决方案

```bash
# 清除缓存重新加载
# 在 Web 界面点击 "🔄 刷新数据" 按钮

# 或重启应用
# 按 Ctrl+C 停止，然后重新运行 python3 run_streamlit.py
```
