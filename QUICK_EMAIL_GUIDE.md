# 📧 模板邮件发送 - 快速使用指南

## 🚀 立即开始

### 1. 访问应用
打开浏览器访问：http://localhost:8501

### 2. 发送邮件的两种方式

#### 方式一：模板生成器页面
1. 在侧边栏选择"📝 模板生成器"
2. 系统自动查找下周主日数据
3. 选择通知模板类型（周三确认/周六提醒/月度总览）
4. 查看生成的模板内容
5. 点击"📧 发送到邮箱"按钮
6. 等待发送完成提示

#### 方式二：模板编辑器页面
1. 在侧边栏选择"🛠️ 模板编辑器"
2. 切换到"🧪 真实测试"选项卡
3. 查看三种通知的实时生成效果
4. 在任意通知下方点击"📧 发送邮件"按钮
5. 等待发送完成提示

## 📧 邮件内容

您将收到包含以下内容的精美HTML邮件：

### 📱 邮件标题
- 周三确认：`【微信群通知模板】周三确认通知 - X月X日`
- 周六提醒：`【微信群通知模板】周六提醒通知 - X月X日`
- 月度总览：`【微信群通知模板】月度总览通知`

### 📋 邮件内容
1. **使用说明**：4步操作指导
2. **通知模板**：完整的微信群通知文本
3. **发送建议**：最佳发送时间提醒
4. **联系信息**：技术支持联系方式

## 🎯 典型工作流程

### 周三确认通知流程
1. **周三下午**：在应用中生成周三确认通知
2. **发送邮件**：点击"发送到邮箱"
3. **查看邮件**：在手机或电脑上打开邮件
4. **复制内容**：从邮件中复制微信群通知文本
5. **发送到群**：粘贴到相应的微信群

### 周六提醒通知流程
1. **周六晚上**：生成周六提醒通知
2. **立即发送**：发送邮件到手机查看
3. **及时通知**：复制到微信群提醒同工
4. **确认回复**：关注同工的确认回复

## ⚙️ 配置检查

### 必需配置
确保 `.env` 文件包含：
```bash
SENDER_EMAIL=jonathanjing@graceirvine.org
EMAIL_PASSWORD=your_gmail_app_password
```

### 可选配置
```bash

```

## 🔧 故障排除

### 常见问题

1. **"邮件发送模块未找到"**
   - 确保已安装所有依赖：`pip install -r requirements.txt`
   - 检查 `src/email_sender.py` 文件是否存在

2. **"邮件发送失败"**
   - 检查 `EMAIL_PASSWORD` 是否正确设置
   - 确认使用Gmail应用专用密码
   - 检查网络连接

3. **"EMAIL_PASSWORD not set"警告**
   - 在 `.env` 文件中添加 `EMAIL_PASSWORD=your_password`
   - 重启Streamlit应用

### 调试步骤
1. 检查应用日志输出
2. 查看Streamlit界面的错误提示
3. 确认邮件配置是否正确
4. 测试网络连接
